VKontakte IM
============

Добро пожаловать в репозиторий исходного кода прототипа XMPP-клиента для ВКонтакте!


Условия
---

Это open-source проект, он участвует в конкурсе <http://vkontakte.ru/pages.php?id=27987544>.

Приглашаем вас присоединиться!
